# DATA COLLECTION CHECKLIST: PROPERTY PASSPORT INITIALIZATION

To initialize your **Digital Twin** and **Rent Ledger**, please provide the following documents:

## 1. Legal & Identity
- [ ] Copy of Trade License (For Companies) or Emirates ID (For Individuals).
- [ ] Copy of Title Deed or Mulkiya for the property.
- [ ] Copy of Management Agreement (if transitioning from another provider).

## 2. Technical Property Details
- [ ] Full list of technical assets (Make/Model of Chillers, Lifts, Fire Pumps).
- [ ] Floor plans / Unit layout (PDF or AutoCAD).
- [ ] Current MEP Maintenance logs (last 6 months).
- [ ] Utility Account numbers (DEWA/ADDC/FEWA).

## 3. Financial & Tenant Data (CSV or Excel)
*We can provide a template for this:*
- [ ] **Unit List:** Unit Number, Type (Studio/1BHK/Retail), Size (SqFt).
- [ ] **Tenant List:** Name, Email, Phone Number, Emirates ID.
- [ ] **Lease Data:** Start Date, End Date, Annual Rent, Payment Frequency (Number of Cheques).
- [ ] **Financial Status:** Current outstanding balance per unit.
- [ ] **Security Deposits:** List of deposits held by the owner.

## 4. Operational Contacts
- [ ] Name and contact of the current on-site watchman/security.
- [ ] Emergency contact list for the building.
- [ ] List of VIP tenants or units with restricted access.
